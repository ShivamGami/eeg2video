🔵 Architecture

7-Stage Pipeline

Multi-path conditioning system transforming raw EEG into semantically coherent video.

1

Feature Extraction

Raw EEG → STFT Spectrogram → Transformer (4L, 8H) → 8×8×512 tensor

EEG → Features
↓
2

Semantic (Text) Path

CLIP-like encoder maps EEG to N×512 text embedding space for cross-attention conditioning

Semantic
↓
3

Visual (ViT) Path

ViT-like encoder extracts M×512 visual embeddings for spatially-aware generation guidance

Visual
↓
4

Dynamic Conditioning

Cross-entropy temporal classifier. Predicts motion dynamics labels for generation conditioning. Acc: 58.27%

Dynamic
↓
5

DANA Latent Conditioning

VAE latents (64×32×32) through DANA Module with conditional noise injection → conditioned latents

Structural
↓
6

Stable Diffusion v1.5 Generation

Conditioning Fusion → SD U-Net (frozen backbone + trainable attn2 layers). Diffusion MSE ≈ 0.0005

Diffusion
↓
7

VAE Decoder → Video

Frozen pretrained VAE decoder maps latent predictions to pixel space. Output: 3 frames at 3 FPS.

Output
